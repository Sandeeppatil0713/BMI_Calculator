This is a BMI Calculator web app. Here's a quick breakdown:

index.html — The UI with two inputs (height in cm, weight in kg), a Calculate button, a results display area, and a BMI weight guide reference table.
style.css — Styles the app with a dark background (#212121), a centered rounded card layout, and a light/frosted-glass look for the container.
script.js — Currently just a skeleton with a form submit listener, but it has a typo ('sumbit' instead of 'submit') and no actual BMI calculation logic yet.
